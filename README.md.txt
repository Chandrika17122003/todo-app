# ✅ To-Do List App

This is a simple and responsive To-Do List application built using **HTML**, **CSS**, and **JavaScript**.

## 🎯 Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Data using local storage
- Mobile responsive design

## 🚀 How to Run

1. Download or clone this project
2. Open the `index.html` file in your browser
3. Start using the To-Do List — no setup required!

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript 
- Local Storage API

## 📱 Responsive

The app works well on both desktop and mobile devices.
